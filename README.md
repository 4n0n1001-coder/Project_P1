# Cyber Security Base 2025 Project 1


## Installation Instructions:

#### 1. Clone the repository
Navigate to your desired folder and run:
```
 $ git clone <github-link>
```

### 2. Install dependencies:
Ensure that you have Python 3.x installed, then install the Django framework:
```
 $ python3 -m pip install django
```

### 3. Navigate to the project root:
```
 $ cd <project>
```

### 4. Run the application
Start the server with the following commands:
```
 $ python3 manage.py runserver
```

### 5. Access the application:
Open your browser and go to `http://127.0.0.1:8000`.
You can log in using the credentials provided in the introduction (bob or alice).
